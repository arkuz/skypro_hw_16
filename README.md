## Описание
Пет проект по типу инстаграм на Flask + Blueprints + Templates + тесты

### Требования к ПО
- Инструменты pyenv и poetry.
- Python 3.10

### Установка и запуск
1. Устанавливаем виртуальное окружение и зависимости проекта:
```bash
poetry install
```

2. Запускаем виртуальное окружение
```bash
poetry shell
```

3. Запускаем проект
```bash
python app.py
```
